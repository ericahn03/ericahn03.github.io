Mortality Prediction Pipeline

Overview

A machine learning pipeline designed to predict mortality rates using Weka and Python. This project applies ensemble models on global health data for high-accuracy forecasts.

Tech Stack

  • Python (Pandas, Scikit-learn)
  • Weka (.arff data integration)
  • Matplotlib / Seaborn for visualization
