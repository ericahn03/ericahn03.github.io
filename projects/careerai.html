Overview

CareerAi is an AI-powered tool that analyzes your resume and finds real-time job listings based on semantic similarity. It streamlines job hunting using Python, Flask, and NLP models to automate resume optimization and job recommendations.

Tech Stack

  • Python (Flask Backend)
  • PDF Resume Parsing (pdfplumber)
  • Natural Language Processing (Semantic Similarity)
  • Frontend: HTML, CSS, JavaScript
  • Live scraping from HiringCafe API
