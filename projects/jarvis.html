Overview

JARVIS is a privacy-focused AI assistant for macOS and Windows. It integrates smart device control, LLM-powered responses, and file management while prioritizing user data privacy.

Tech Stack

  • Python (Flask, LangChain)
  • macOS/Windows compatibility
  • Speech recognition and OpenAI integration
